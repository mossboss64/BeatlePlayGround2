The Beatles Playground

A responsive, interactive Beatles exploration website built with vanilla HTML, CSS, and JavaScript. This project was developed as part of a front-end web development learning experience, with an emphasis on visual storytelling, user interaction, and iterative UI refinement through experimentation.

Features

Interactive Album Experience Click album covers to explore Beatles albums, trigger dynamic color themes, and play song previews with custom play and pause controls.

Album-Themed Color Switching Each album applies a unique color palette inspired by its original cover art, creating an immersive and cohesive visual experience.!
Responsive Design Layout adapts smoothly for desktop, tablet, and mobile devices using flexible layouts and media queries.

Animated UI Elements Includes hover effects, album scaling, flip-card interactions, and smooth transitions to enhance usability and engagement.
